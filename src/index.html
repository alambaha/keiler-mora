Libros Favoritos de Alicia
aliciatorres.com

Se trata de una aplicación web para mostrar los libros favoritos de Alicia, los cuales pueden ser filtrados por el título, autor o género literario. El proyecto personal fue desarrollado con React, una libería de JavaScript para elaborar interfaces de usuario. Junto al manejador de eventos Redux, la aplicación muestra la información de los libros previamente cargados en el backend desarrollado en GraphQL, llevándolas a la Single-Page Application.

Diagnóstico L'Oréal
diagnosticolorealprofessionnel.com

La empresa deseaba ofrecer una promoción durante un evento presencial a través de una aplicación web. Los usuarios debían registrarse y responder una encuesta acerca del cuidado de su cabello y, en base a sus respuestas, la aplicación mostraría un listado de productos recomendados. Para ello, se desarrolló una REST API usando PHP en el backend, manteniendo comunicación con el manejador de base de datos MySQL.
